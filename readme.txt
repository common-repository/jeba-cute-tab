Jeba Cute Tab is an awesome Tab, super lightweight plugin for your wordpress website post Tab. 

== Description ==

This plugin will enable awesome Tab in your wordpress theme. You can use the jeba cute Tab via shortcode in everywhere you want, even in theme files.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use shortcode in page, post or in widgets.
4. If you want use the jeba cute Tab in your theme php, Place `<?php echo do_shortcode('[jeba_Tab]'); ?>` in your templates.
5. In the [jeba_Tab] shortcode you can change post_type="", category="", count="" . Here count is total post. 
Shortcodes
6. On the editor show 'jeba' tiny MCE buttons.
<strong>Use Jeba Cute Tab shortcode</strong>
<pre>[jeba_Tab]</pre>

<strong>Here can change</strong>
<pre>[jeba_Tab post_type="" category="" count=""]</pre>

== Frequently Asked Questions ==
= How can change post_type="" category="" count=""? =
In "post_type" you can use "post", "page" and other "custom_post_type". In "category" you can use your post category to tab category based. In "count" can use total how many post want to be tab.
